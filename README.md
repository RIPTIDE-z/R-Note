# R-Note

## 项目说明

- 实践分布式事务控制的项目
    - 后端使用`Spring Boot` + `Druid` + `MyBatis` + `Atomikos` + `MySQL`
    - 前端使用`Qt Widgets`
    - 使用`RESTful API`标准、`APIfox`软件辅助设计接口
- 项目模板为带有简单 **用户注册/登录**，笔记 **历史查看/回滚** 的代码笔记软件
